# 🎲 Sorteador de Números

Projeto desenvolvido com HTML, CSS e JavaScript que permite sortear um número aleatório entre dois valores informados pelo usuário.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 📋 Funcionalidades

- Inserir valor mínimo
- Inserir valor máximo
- Gerar número aleatório entre os valores informados
- Interface simples e intuitiva

## 📸 Preview

O usuário informa um número inicial e um número final e, ao clicar em **Sortear**, recebe um número aleatório dentro do intervalo escolhido.
